1. Do the users spend more time on the new landing page than the existing landing page?¶

In [27]:
# First, I need to seperate the data based on which landing_page was used
new_data = data[data['landing_page'] == 'new']
old_data = data[data['landing_page'] == 'old']

# This question requires a Two Sample Independant t-test
from scipy.stats import ttest_ind
t_stat, p_val = ttest_ind(new_data['time_spent_on_the_page'], old_data['time_spent_on_the_page'],alternative='greater')
print(f't-statistic: {t_stat}')
print(f'p-value: {p_val}')

Step 1: Define the null and alternate hypotheses¶

$H_0$: $\mu_{new}$ ≤ $\mu_{old}$

$H_a$: $\mu_{new}$ > $\mu_{old}$

Step 2: Select Appropriate test¶

Two-sample independant t-test (mean)

Step 3: Decide the significance level¶

$α = 0.05$

Step 4: Collect and prepare data¶

In [29]:

Step 5: Calculate the p-value¶

In [30]:
# import the apporpriate test and calculate p-value
from scipy.stats import ttest_ind

print(f't-statistic: {t_stat:.4f}')
print(f'p-value: {p_val:.4f}')

t-statistic: 3.7868

Step 6: Compare the p-value with $\alpha$¶

In [31]:
# compare p-value with significance level
alpha=0.05
print(f'p-value: {p_val:.4f}')
if p_val <= alpha:
    print('Reject the null hypothesis')
else:
    print('Fail to reject the null hypothesis')

p-value: 0.0001
Reject the null hypothesis

Step 7: Draw inference¶

When we reject the null hypothesis that does not necessarily mean the alternative hypothesis is true. However, since we are rejecting the null hypothesis, this does support the alternative hypothesis. This means there is support for the alternative hypothesis that more time is spent on the new page than the old page.

ANOVA Test (means)

Step 3: Decide the significance level¶

$\alpha = 0.05$

Step 4: Collect and prepare data¶

In [49]:
